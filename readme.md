# Elephant Comerce

Реализованна система авторизации, пользователь может создать учётную запись, удалить (soft), войти в аккаунт и выйти из него.

У пользователей есть 4 уровня доступа:
* __Гость__, может просматривать стандартные товары и оставлять комментарии, по-умолчанию все пользователи регистрируются как гости; 
* __VIP__, как и гость, но также видит премиальные товары; 
* __Продавец__, то же самое что и гость;
__Админ (staff)__, имеет доступ к админской панели, добавляет товары, выдаёт права пользователям, восстанавливает "удалённые" аккаунты

> Для запуска проекта необходим файл `config.py` в одной папке с `settings.py`, он должен содержать `MY_SECRET_KEY` от проекта
